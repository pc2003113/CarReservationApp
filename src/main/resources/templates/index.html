Car Reservation App

The Car Reservation App is a web based vehicle rental site for customers to rent a vehicle

This site will have two types of users, namely, the Customer and the Admin

In the case of the customer, in order to use this app, the customer will have to first register, create a profile and login to the app for making the reservation of the selected car

The Admin will use this site to manage users and vehicle inventory

The following are are the available features of this website for the cutomer:-

  1. Customer Registeration
  2. Customer Login
  3. Make Reservation
  4. View Reservations
  5. Logout

The following are are the available features of this website for the Adminstrator:-

  1. View Registered Users
  2. Add Cars
  3. View Car Inventory
  4. Logout
